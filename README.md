## Getting Started

### Prerequisites
- Node.js (version 14 or higher)
- npm or yarn package manager

### Installation

1. **Clone or download the project files**

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm start
   ```

4. **Open your browser** and navigate to `http://localhost:3000`

### Building for Production

To create a production build:

```bash
npm run build
```

The build files will be generated in the `build` directory.

## Usage Guide

### 1. Select a Template
- Browse through the categorized template library
- Click on any template to select it
- The preview will update automatically

### 2. Customize Your Email
- Switch to the "Customize" tab
- Edit the subject line, header text, and body content
- Change the primary brand color using the color picker
- Add your company logo URL
- Customize the call-to-action button
- Update footer information
- Toggle elements on/off as needed

### 3. Export Your Template
- Go to the "Export" tab
- Copy the HTML code to clipboard for immediate use
- Download as an HTML file for later use
- View the raw HTML code if needed

### 4. Use in Email Campaigns
- Paste the HTML into your email marketing platform
- Upload the HTML file to your email service provider
- Test across different email clients before sending
